= Employee Management REST API
Yuvasri S

This is a Spring Boot REST API developed during a Forage internship.
It allows managing employees using CRUD operations (Create, Read, Update, Delete).

== Technologies Used
- Java
- Spring Boot
- Gradle / Maven
- REST API
- MySQL / H2 Database

== Features
- Add new employees (POST)
- View employee details (GET)
- Update employee data (PUT)
- Delete employee records (DELETE)

== How to Run
1. Open project in IntelliJ IDEA
2. Run RestServiceApplication.java
3. Test API endpoints using Postman

== API Endpoints (Example)
- GET /employees
- POST /employees
- PUT /employees/{id}
- DELETE /employees/{id}
